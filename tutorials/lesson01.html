Objetivo

Seu objetivo agora é levar nosso personagem Clóvis até o outro lado da estrada para que continue seu caminho até a fazenda.

Documentação - Introdução

Você pode reparar que o editor de texto já vem preenchido com algum código.
Esse código é um esqueleto necessário para que tudo funcione de acordo, mas não se preocupe com os class, def, self e etc.. por enquanto.
Esse são conceitos importantes na programação, mas não são necessários agora.

O único que você precisa estar atento é o play_turn (do inglês jogue_turno).
play_turn é uma função (funções são como comandos ou ações) e toda lógica do nosso jogador deve ser escrita dentro dela.
No Python denominamos que estamos escrevendo código dentro de uma função ao colocar uma identação, ou seja, 4 espaços em branco a mais (ou 1 tab).
Mas claro que você sabe disso pq você fez o tutorial rápido de Python, certo?

Então você precisa escrever seu código dentro da função play_turn para o Clóvis andar.
"Ok, isso ficou claro, mas como?" você deve estar se perguntando.
Para responder isso, vamos com mais documentação:

Documentação - Player

Repare que a função play_turn recebe 2 argumentos: self e player, mas vamos ignorar o self por enquanto.
O player (do inglês jogador) é a representação do seu jogador, o Clóvis, e todos os comandos de ação vão partir dele, por exemplo o comando wait (do inglês, esperar):

player.wait()

Faz o Clóvis esperar esse turno e não fazer nada.
ps: Se você não sabe porque colocar esses () após o wait, é porque wait é um comando dentro do player, comandos (ou funções) são sempre chamados utilizando parênteses depois do nome, mas explicaremos mais depois.

Aqui está uma lista inicial de funções disponíveis no player:

player.moveUp() - O jogador anda uma casa para cima.
player.moveRight() - O jogador anda uma casa para direta.
player.moveDown() - O jogador anda uma casa para baixo.
player.moveLeft() - O jogador anda uma casa para esquerda.
player.wait() - O jogador não faz nada e espera.

Mas atenção!

Você só pode usar um movimento físico durante o turno. Movimentos físicos são aqueles que tem algum tipo de movimentação ou força física, como andar.
Caso você use mais de um movimento físico durante o turno, ele será ignorado, aparecerá um aviso no console e você terá pontos descontados por isso.
